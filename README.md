# Alien Stage Music Archive

My first project ever! A fanpage/directory website with a short summary and song archive of the Alien Stage(ALNST) web series, built using HTML, CSS, and JavaScript.

## Features
The homepage includes a spotify player of all songs released by the ALNST singers, including the sub-songs from individual characters. 
Next to the spotify player, the image opens a song directory/index that has buttons for all the official rounds. 
Below this section is a slider with songs and videos connected to the series, despite not being specified as rounds.

### Song-directory
- Lyrics to each song
- GIF at the top takes you to a youtube link
- Underlined names of the artists open the song in spotify.
- Video embed of the round at the end of each song's section.
